QMGrid Theme Demo

Demonstrating light and dark theme support with CSS variables

Theme Configuration

The table now defaults to light theme regardless of system preferences. You can control the theme programmatically:

  • new QMGrid('#table', { theme: 'light' }) - Set initial theme
  • grid.setTheme('dark') - Switch to dark theme
  • grid.toggleTheme() - Toggle between themes
  • grid.getTheme() - Get current theme
